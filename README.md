# Clone with submodules

```
git clone --recurse-submodules https://github.com/thisarasamuditha/iwb210-hashmap
```

# Start App

```
docker compose up --build
```

> visit localhost:4000

# Stop App

```
docker compose down
```

## On slow connections

> build backend seperately

```
docker build backend
```

then run;

```
docker compose up
```
